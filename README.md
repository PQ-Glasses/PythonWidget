## GetWifiPassword

## SplitGIF

![皮卡丘](https://github.com/PQ-Glasses/PythonWidget/blob/master/SplitGIF/pq.gif)

将GIF动图每帧分开后，程序会新建一个defaule文件夹，并将图片放入该文件夹中，文件名称会以图片原名加动图的顺序命名。

使用方式：

```bash
python splitGIF.py filename.gif
```
